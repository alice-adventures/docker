Welcome to Alice

Alice Adventures for Learning and Inspiring Coding Excellence

Alice is framework that empowers developers and enthusiasts to explore the wonderland of Ada programming.

Alice is its early stages of development. Dive into the collection of enchanting problem sources, engage in collaborative tea parties, and unleash your coding creativity. Come to explore the rabbit holes of possibilities!

Alice in Dockerland

This Docker container is a fully functional installation of Alice: it has everything you need to start exploring Alice and to keep working with it in the future. New content updates can be applied seamlessly.

Current Status

  • Current supported problem source is Project Euler.

  • Support for Checkio is under development.

  • There are plans to support Advent of Code, Codingame and Rosseta Code really soon.

  • Any additional Problem Source you would like to have, please let us know.

Develop in Alice

Implement existing functionality using different libraries.

Develop support for new Problem Sources by adding new interfaces that are better suited to them.

Implement new data structures and algorithms that Participants can use to solve programming challenges.

Create code sharing guidelines for Participants to follow when sharing their code on Alice.

Participate in Alice

Solve Problem challenges and explore new solutions by using parallelism, parametrize algorithm behavior or changing initial conditions.

Learn Ada by example: understand what other participants have developed and improve the designs or implementations shared by others.

When there is GUI support for a Problem Source, create graphical explanations tow show how your algorithm works, the search space is explored or whatever you can imagine.
